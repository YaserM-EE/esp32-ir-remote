# esp32-ir-remote
Embedded C++ firmware and KiCad hardware design for a custom ESP32 universal infrared remote.

**INTRODUCTION**
This repository documents the hardware design and embedded firmware for a custom universal infrared remote. Built around the ESP32 WROOM microcontroller, the project utilizes C++ and the IRremoteESP8266 library to capture, decode, and transmit standard 38kHz IR protocols. The final hardware iteration includes a custom-routed printed circuit board designed in KiCad.
